# ventmar-Codex

Landing y onboarding multi-step para Ventmar Argentina, construido con Next.js, Tailwind CSS y Supabase.

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Supabase

## Variables de entorno

Crea un archivo `.env.local` con:

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

## Desarrollo

```bash
npm install
npm run dev
```

## Produccion

```bash
npm run build
npm run start
```
